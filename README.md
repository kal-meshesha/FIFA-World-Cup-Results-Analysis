# FIFA-World-Cup-Results-Analysis
## Tableau Dashboard | Sports Analytics | Historical Performance Analysis | Data Visualization
### Executive Summary

Football analysts and fans often struggle to extract meaningful insights from historical World Cup data due to its size and complexity.

Using Tableau, I built an interactive dashboard analyzing historical FIFA World Cup match results, team performance, scoring trends, and tournament outcomes.

The analysis highlights dominant teams, goal-scoring trends, and performance patterns across tournaments â€” transforming raw match data into clear, visual insights.

### Business Problem

The FIFA World Cup spans decades of matches, teams, and outcomes. However, raw match-level data does not clearly answer key questions such as:

Which countries perform best historically?

How have goal-scoring trends changed over time?

Which teams consistently reach later tournament stages?

Are there patterns in host country performance?

This project converts historical match data into structured, visual analysis.

### Methodology
#### 1. Data Preparation

Cleaned World Cup results dataset

### Key Insights
